## Project template (uv + notebooks + src/)

This repository is a notebook-centric project template that uses **uv** for environment management.

### Prerequisites

- **bash**
- **uv**
  - macOS (Homebrew): `brew install uv`
  - macOS / Linux / WSL (installer): `curl -LsSf "https://astral.sh/uv/install.sh" | sh`

### Initialize a new project (recommended)

Run the bootstrap script from the repo root:

```bash
bash scripts/init_project.sh
